INSTALLATION INSTRUCTIONS:

Local Development:
1. Place files in web server directory (htdocs for XAMPP/MAMP)
2. Start Apache and MySQL servers
3. Run setup_database.php to create database
4. Access application via http://localhost/blog-application/

Hosting Deployment:
1. Upload all files to web server
2. Create MySQL database through hosting control panel
3. Import database_schema.sql or run setup_hosting.php
4. Update database credentials in config/database.php if needed
